# Actor 5: System
As the system, I want to automatically calculate daily calorie and macronutrient goals based on user profile data so that recommendations are tailored to individual needs.✔️
As the system, I want to process food images using machine learning to identify food items and estimate calories so that users can log meals effortlessly.✔️
As the system, I want to generate personalized meal recommendations using collaborative filtering and content-based algorithms so that suggestions match user preferences.✔️
As the system, I want to send daily notifications reminding users to log meals so that tracking consistency is maintained.✔️
As the system, I want to automatically fetch recipe data from external APIs (Spoonacular) so that the recipe database stays up-to-date.✔️
As the system, I want to analyze user dietary patterns over time so that I can adapt future recommendations based on behavior.✔️
As the system, I want to perform nightly backups of user data so that information is protected against data loss.✔️
As the system, I want to generate weekly and monthly dietary reports with charts and visualizations so that users can track progress.✔️
As the system, I want to handle the cold-start problem for new users by asking preference questions so that initial recommendations are relevant.✔️
As the system, I want to monitor API rate limits and implement caching strategies so that external service costs are minimized.✔️
As the system, I want to automatically scale meal portions based on user calorie goals so that recommendations match individual caloric and nutritional requirements.✔️
As the system, I want to adjust recommendations when the user adjusts goal so that results stay relevant.✔️
As the system, I want to detect inactivity so that I can trigger a motivational notification.✔️
As a system, I want to send email verification to new users so that accounts are authenticated.✔️
As a system, I want to automatically renew premium subscriptions so that users have uninterrupted service.✔️
As a system, I want to send payment receipts to users so that they have transaction records.✔️
As a system, I want to automatically delete accounts marked for deletion after 30 days so that data privacy is maintained.✔️

# Actor 6: Nutritionist
As a nutritionist, I want to register on NutriTrack with my professional credentials (license number, specialisation, institution) so that users and admins can verify my qualifications and trust the advice I provide.
As a nutritionist, I want access to a dedicated dashboard showing my active clients, scheduled consultations, and pending messages so that I can manage my workload efficiently within the app.
As a nutritionist, I want to view the dietary logs, calorie history, macro breakdowns, and goal progress of users assigned to me so that I can give informed, data-driven consultations.
As a nutritionist, I want to conduct real-time text-based consultation sessions with premium users so that I can provide personalised dietary guidance directly within the app.
As a nutritionist, I want to author and publish nutrition tips, articles, and advice cards to the app's content library so that both freemium and premium users benefit from professional guidance at scale.
As a nutritionist, I want to set my availability schedule and accept or decline incoming consultation requests from premium users so that I can manage my time and client load effectively.
As a nutritionist, I want to leave written feedback comments on a client's logged meals or weekly diary so that I can guide clients between live consultation sessions without requiring a scheduled call.
As a nutritionist, I want a public-facing profile page within the app showcasing my specialisations, credentials, client testimonials, and sample tips so that prospective and freemium users are encouraged to upgrade to premium for personal consultations.
As a nutritionist, I want to view analytics on my client engagements — including consultation counts, client goal achievement rates, and content engagement metrics — so that I can measure my impact and improve my service over time.
